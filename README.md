# GhostIp

[![Ruby](https://img.shields.io/badge/Ruby-3.0%2B-red)](https://www.ruby-lang.org)
[![Tor](https://img.shields.io/badge/Tor-Required-blue)](https://www.torproject.org)


GhostIp changes your IP address via the Tor network:)

https://github.com/monji024/GhostIp/raw/main/Ghost.gif

## Quick Installation

```bash
git clone https://github.com/monji024/GhostIp.git
cd GhostIp
chmod +x install.sh
sudo ./install.sh
```

## How to run!!!

```ruby
sudo ruby main.rb
```
or 
```bash 
sudo su 
```
```ruby 
ruby main.rb
```

## Port 9050 in use :

```bash 
sudo fuser -k 9050/tcp
```
